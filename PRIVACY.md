# PRISM 棱镜 隐私说明

**最后更新：2026 年 6 月**

PRISM 棱镜（以下简称「本扩展」）是一款本地运行的 Chrome/Edge 网页资源提取扩展。本扩展不提供账号系统，不采集、上传或出售任何个人信息。本隐私说明旨在向用户透明地介绍扩展的数据处理方式。

---

## 数据存储

本扩展仅使用浏览器内置的 `chrome.storage` API 在本地保存数据，包括：

- 用户选择的主题（皮肤）偏好
- 用户的筛选设置（慢加载阈值、默认 Tab、导出格式等）
- 上次提取的资源数据（仅在内存中，关闭弹窗即清除）

**以上数据仅保存在用户自己的浏览器中，不会传输到任何外部服务器。**

- 使用 `chrome.storage.sync` 保存的主题和设置，会通过用户的 Google 账号跨设备同步（由 Chrome 浏览器管理，本扩展不访问）
- 卸载扩展或清除浏览器扩展数据后，本地数据将被永久删除

---

## 网络请求

本扩展仅在以下场景发起网络请求：

1. **提取当前页面资源**：读取当前页面已加载的资源信息（通过 DOM 和 Performance API），不发起额外请求
2. **批量导出 ZIP**：用户主动触发导出时，通过 Service Worker 代理下载用户选中的资源
3. **单个下载资源**：用户主动点击下载按钮时，通过 Chrome Downloads API 下载

**本扩展不会主动向任何第三方服务器发送用户数据。**

---

## 权限用途

| 权限 | 用途 |
|------|------|
| `activeTab` | 访问当前活动标签页，提取资源信息 |
| `scripting` | 注入 Content Script 到目标页面 |
| `storage` | 保存用户主题偏好和设置 |
| `contextMenus` | 注册右键菜单（提取图片/链接/选区） |
| `downloads` | 用户主动下载资源时使用 |
| `<all_urls>` | 代理下载跨域资源（仅在用户触发导出时） |

本扩展仅申请运行所必需的最小权限集，不会申请超出功能需要的权限。

---

## Content Script

本扩展的 Content Script 仅在以下情况运行：

- 用户点击扩展图标打开弹窗时
- 用户通过右键菜单触发提取时

Content Script 仅读取当前页面的 DOM 结构和 Performance API 数据，**不会修改页面内容、注入广告或追踪代码**。

---

## 第三方服务

本扩展当前不接入任何第三方分析、广告、追踪或 AI 服务。
