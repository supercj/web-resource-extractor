# PRISM Lens - Web Resource Extractor

> 🔷 A powerful web resource extraction tool — extract, classify, visualize, and batch export all web resources with one click

[中文文档](README.md) | **English**

---

## ✨ Features

### 🎯 Core Features
- **One-Click Extraction** - Extract all page resources (images, scripts, styles, fonts, videos, audios, links)
- **Smart Classification** - Automatically categorize resources by type
- **Batch Operations** - Select multiple resources for batch download or export
- **ZIP Export** - Package selected resources into a ZIP file for download

### 📊 Advanced Features
- **Waterfall Chart** - Visualize resource loading timeline with performance data
- **Statistics Panel** - Display resource count, size distribution, and loading time
- **Performance Analysis** - Highlight slow-loading and large resources
- **Third-Party Detection** - Identify external domain resources

### 🎨 User Experience
- **4 Built-in Themes** - Default, Dark, Ocean, Sakura
- **Real-time Search** - Filter resources by URL or domain
- **Advanced Filters** - External resources, large files, slow loading
- **Extension Filter** - Filter by file extension

### 🚀 Dynamic Monitoring
- **SPA Support** - Monitor route changes in Single Page Applications
- **Lazy Loading Detection** - Capture dynamically loaded resources
- **DOM Mutation Monitoring** - Detect new resources added to the page
- **Network Monitoring** - Track resources loaded via fetch/XHR

## 📦 Installation

### Method 1: Install from Edge Add-ons Store (Recommended)
Search for "PRISM Lens" in [Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons)

### Method 2: Install from Source
1. Download or clone this repository
2. Open Edge browser and navigate to `edge://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the extension directory

---

## 🎯 Usage

### Basic Usage
1. Navigate to any web page
2. Click the extension icon
3. Resources are automatically extracted and displayed by category
4. Select resources and click "Download Selected" or "Export ZIP"

### Advanced Features
- **Statistics**: Click the 📊 icon to view resource statistics
- **Waterfall Chart**: Click the 📈 icon to see the loading timeline
- **Theme Switch**: Click the 🎨 icon to change themes
- **Search Filter**: Enter URL or domain in the search box
- **Extension Filter**: Click "📁 Extension Filter" to filter by file type
