## ⚡ 快速开始

### 1. 创建 ZIP 包

**Windows:**
```bash
# 双击运行
prepare-edge-publish.bat
```

**或手动：**
选中文件 → 右键 → 发送到 → 压缩文件夹 → 重命名为 `prism-v1.1.0.zip`

### 2. 准备截图

访问测试网站（如 bing.com）截取 5 张图：
1. 主界面
2. 瀑布图
3. 筛选功能
4. 统计面板
5. 主题切换

尺寸：1280x800（推荐）或 640x400（最小）

### 3. 注册账号

访问：https://partner.microsoft.com/dashboard/microsoftedge
- 使用 Microsoft 账号登录
- **免费注册**（无需费用）

### 4. 提交扩展

1. 创建新加载项
2. 上传 ZIP 包
3. 填写产品信息
4. 上传截图
5. 提交审核
