## 二、商店描述

### 简短描述（Chrome ≤132 字符 / Edge ≤256 字符）

> 网页资源透视镜 — 一键提取、分类展示、瀑布图分析、批量导出网页资源，4 套主题皮肤可选。

### 详细描述（Chrome Web Store）

```
🔷 PRISM 棱镜 — 网页资源透视镜

像棱镜分解光谱一样，PRISM 将网页分解为各类资源。开发者、设计师、前端工程师必备的网页资源分析工具。

【核心功能】

🔍 全面提取
自动识别页面中的图片、JS 脚本、CSS 样式、字体、视频、音频、超链接和第三方资源。支持 <img>、CSS 背景图、<picture>、SVG、favicon、<video poster> 等多种来源。

📁 分类展示
8 大分类 Tab：图片 / 脚本 / 样式 / 字体 / 视频 / 音频 / 链接 / 第三方，资源一目了然。

📊 统计面板
查看资源总数、总大小、页面加载时间，以及按类型分布的柱状图。

📈 加载瀑布图
可视化资源加载时间线，快速定位慢加载资源（超过阈值自动标红）。

🔎 智能搜索
关键词搜索 URL 和域名 + 后缀筛选（如只看 .png、.js）+ 外部资源 / 大文件 / 慢加载过滤。

📥 单个下载
点击即可下载任意资源。

📦 批量导出
勾选多个资源后一键打包为 ZIP 下载。跨域资源通过 Service Worker 代理，自动处理同名文件。

🖱️ 右键菜单
右键页面快速提取：所有图片、所有链接、选中区域资源、页面全部资源。

🎨 4 套皮肤
☀️ 默认（清新亮色）/ 🌙 暗黑（护眼）/ 🌊 海洋蓝 / 🌸 樱花粉，一键实时切换。

🔒 隐私优先
所有数据仅存储在浏览器本地，不采集、不上传、不追踪。

【适用场景】
- 前端开发：分析页面资源加载性能
- 设计师：批量提取网页图片素材
- SEO 优化：检查第三方资源和外部链接
- 安全审计：识别页面加载的所有外部资源

开源地址：https://github.com/your-username/prism
隐私说明：https://github.com/your-username/prism/blob/main/PRIVACY.md
```

## 七、权限说明（审核必备）

审核人员会检查权限合理性，以下是每个权限的用途说明：

| 权限 | manifest 中的声明 | 审核说明 |
|------|------------------|---------|
| `activeTab` | 访问当前标签页提取资源 | 核心功能必需，用户点击时才激活 |
| `scripting` | 注入 Content Script 读取页面资源 | 核心功能必需 |
| `storage` | 保存用户主题和设置偏好 | 核心功能必需 |
| `contextMenus` | 注册右键菜单 | 增强功能 |
| `downloads` | 用户主动下载资源 | 用户主动触发 |
| `<all_urls>` | 代理下载跨域资源 | 仅在用户触发 ZIP 导出时使用 |

> Content Script 仅读取 DOM 和 Performance API，不修改页面内容。

## 九、版本更新流程

1. 修改 `manifest.json` 中的 `version` 字段
2. 更新 `README.md` 中的更新日志
3. 重新打包 `.zip`
4. 分别在 Chrome Web Store 和 Edge Add-ons 上传新版本
5. 填写更新说明
6. 提交审核
